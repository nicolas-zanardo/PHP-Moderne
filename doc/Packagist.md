# Packagist

<https://packagist.org/>

Permet d'installer des pakadges PHP avec composer.

```JSON
 // Exemple
 {
    "require": {
        "vendor/package": "1.3.2",
        "vendor/package2": "1.*",
        "vendor/package3": "^2.0.3"
    }
 }
```

---

## Installation de nesbot/carbon

<https://packagist.org/packages/nesbot/carbon>

```Bash
    composer require nesbot/carbon
```
